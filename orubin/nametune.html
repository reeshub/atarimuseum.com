Name That Tune - 1986 Bally Sente Inc.









This was a hard game to do. Some GREAT music work by Richard Green, a contractor musician we hired to arrange for us, and some great hardware work to add even more memory. Trying to make the game play like the show was hard, but we (the team) pulled off a decent game. Should have been a bar game, they did not sell it correctly, but it was fun.
