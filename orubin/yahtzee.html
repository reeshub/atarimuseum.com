Yahtzee - 1980 Atari Inc.
for the Atari 8-bit personal computers

I am not sure if this game was ever released.

This was actually written in FORTH (Thanks Steve Calfee), and I did this to get myself an Atari 800. I wanted Atari to pay for it, so I helped set up a deal where if we did a game for the consumer computer, we would geta unit. My game worked, was fun. BUT, we did not have the rights to Yahtzee so we could not ship it. It may have gone out finally under the name "5-Dice", as that was the final version.
