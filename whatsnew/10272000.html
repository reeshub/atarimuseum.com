October 27, 2000 Update:
 
 

CatBox Portable Diagnostic tool added to Atari Coinop/Arcade Section
Atari ROM and Source Code page added to Atari Archives section.  (This should be a good resource for anyone interested in the inner workings of some of Atari's various computers & video games.   Also the Binary files have been disassembled and saved as text files for easy use and reference.)
 
 

Retro Reproductions is now:  Retro-Engineering.     Project Schedules:
 

   I've been getting more and more email asking about other projects that have been mentioned in the past, here is a schedule of upcoming hardware:
 

1. Atari 7800 Computer Keyboard (Spring 2001, CPU's located and ordered, basic PCB layout completed, finalizing keyboard case purchase) (Will Debut at Phillyclassic in April)   Keyboard will include OS source code and copy of the original technical document to allow anyone who is interested to write additional software for the 7800 Computer Keyboard.

2. XE-2-ST 3.5" disk drive interface (allows the use of Atari SF354 & 314 disk drives on any Atari 8bit computer) schematic completed, sourcing parts and XEP80/SX212 cases for use.   (Spring 2001:  Will debut at CCAG show in Ohio in June)

3. CX870 IDE Interface module for Atari 800 Computers only, Slot#3.   Simple Interface module, based on myide initial design, will ship with drivers, cables and instructions.   Special edition 870 Hard drive will be available to first 10 orders (Modified Atari 810 case with new front, built in power supply and IDE hard drive, compliments your Atari 800 and gives a complete look to the system)    (Spring 2001: Will debut at CCAG show in June)

4. NewBUS:   Thanks to Michael St. Pierre.    ISA Bus for Atari 8-bit computers.    (Completed Prototype being reviewed, more news to come, possible Summer 2001 release.  More news to come after 1st of year.)   This will be a public domain project, I will make some units, but also all technical data and information will be released into public domain for all to reference and use.    Many other technical documents will be released into public domain thanks to Michael St. Pierre for releasing them to me for public domain use.
