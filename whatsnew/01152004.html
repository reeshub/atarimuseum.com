AHS - Site News:   January 15, 2004

 

Atari Archives Server update:   Tech Docs Section --

The Archives server where the main store of nearly 700mb's of Atari Historical Data is kept has been updated with additional files today.

New additions are to the Atari 7800 section and include:

The Source Code for High Score Card Activation.

Schematic for the Atari 7800 Computer Keyboard

Schematics for the Atari 7800 Trakball

Schematic for the Atari 5200-7800 Adapter

(It should be noted, the source document that was scanned in for the 5200-7800 is terribly out of focus and best efforts were made to sharpen the document, however viewing quality is difficult.)
