Atari History Site: January 28, 2001 Update
 
 

Great news for the Jaguar 64 programming community.    Glenn Bruner has done it!!!   He has located on an old Syquest removable hard disk that came out of Atari, a file that appears to successfully bypass the Atari Jaguar 64 encryption system when appended to a non-encrypted game file.  This discover should help open the way for many of the homebrew and hobbyist games to be placed into rom's and used on unmodified Jaguar 64 game consoles.
 

To Glenn...  as always, nice work!!!
