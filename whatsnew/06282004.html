AtariMuseum - Site News:   June 28, 2004

 

Atari Archives Server update:   Tech Docs Section --

The TechDocs section over on the Archives server has received a minor upgrade with a new drop down menu'ing system.   A great deal of new content is being prepared for the TechDoc's section, so these new drop down menu's will make accessing the files more easy.

A new section has been opened under the Computers Section on the main WWW server called Atari "SWEAT" Tools.     Sweat stood for Software Editors and Tools.   These were the actual internal developer tools used by Atari to create applications and games.   Hopefully they will come in handy for those looking to program on the Atari 8bits or to develop games for the 5200 Super System.
