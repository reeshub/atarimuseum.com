"SWEAT" was Atari's internal answer to its programmer staff that had complained over the years that there needed to be a set of tools to streamline and simplify basic tasks while developing new applications and games for the Atari computer systems.    

These tools were in development for several years before finally being released to the programming staff.    While not great games in of themselves, ET Phone Home and Superman III were two games that utilized these new tools.    

In the Tramiel years of Atari, Rob Zydbel delivered a set of these tools to John Skruch for use by the XE Game System programming staff...   In fact Rob wrote John Skruch a "Dear John" letter, jokingly stating "he always wanted to start a letter like that."
