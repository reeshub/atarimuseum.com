Atari 1650XLD  

 

 

  

 

 

 

 

 

The Atari 1650XLD was to be a 68000 based computer.  As one engineering log noted:  It was to be a Sexy machine.    Atari planned to have a Windows Operating System on it.    Talk among the various groups was the idea of either using the VISI ON graphical windowing OS or Atari completing its own in-house custom OS codenamed "Snowcap."

Atari Engineer Omar Gadir wrote a memo on this new "Lisa type machine" that Atari needed to develop.
